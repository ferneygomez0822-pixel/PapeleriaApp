✅ Características del Sistema

	Aplicación de escritorio desarrollada en Java Swing.
	Diseño modular organizado en paquetes (DAO, modelos, UI, database).
	Gestión de información en base de datos mediante consultas SQL.
	Interfaz gráfica amigable para manipular datos de la papelería.
	Implementación de MVC por capas, lo que mejora el mantenimiento.
	Procesos CRUD completos (crear, leer, actualizar, eliminar).
	Manejo de conexión a la base de datos mediante JDBC.
	Sistema adaptable a las necesidades de cualquier papelería.

✅Objetivo General

Desarrollar una aplicación de escritorio que permita gestionar de manera eficiente los principales procesos administrativos de una papelería.

✅Objetivos Específicos

•	Implementar un módulo para la gestión de productos, clientes, usuarios y categorías.
•	Diseñar una interfaz gráfica que facilite la interacción con el sistema.
•	Integrar una base de datos que almacene de forma segura la información.
•	Crear un sistema de facturación básico para registrar ventas.
•	Aplicar una arquitectura organizada que permita su ampliación futura.
